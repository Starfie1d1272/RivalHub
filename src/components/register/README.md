# components/register

报名系统 UI 组件。

## 规划组件

| 组件 | 说明 |
|---|---|
| `RegistrationForm` | 完整报名表单（React Hook Form + Zod，多步骤） |
| `PositionCapBar` | 各位置实时名额进度条（接收 Realtime 推送） |
| `ScreenshotUpload` | 截图上传组件（客户端直传 Supabase Storage，进度条） |
| `RegistrationSuccess` | 报名成功页组件（含 Magic Link 提示） |
| `PositionSelect` | 位置下拉选择（满员自动置灰） |
